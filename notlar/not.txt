Geliştirme bilgisayarı (siz — kod + sürüm)

	1-Kod üzerinde çalışın (Cursor / VS Code).
	2-GitHub’a göndermek için: kaydet-ve-githuba-gonder.bat
		git add → commit mesajı → git push origin main
	3-Müşterilere / işyerine kurulum paketi (EXE + ZIP + release) çıkarmak için: release-all.bat
	4-Yerel musteri-paketi klasoru (C:\musteri-paketi veya musteri-paketi\): musteri-paketi-olustur.bat
	   (release ile birlikte: musteri-paketi-olustur.bat release)
		Sürümü artırır, EXE + ZIP üretir, GitHub Release’e yükler, main + tag dener.
	4-token.txt (veya local-release.bat) sadece bu makinede dursun; Git’e commitlenmez (.gitignore).

İşyeri / kasa bilgisayarı (güncelleme + açma)
	1-ilk kurulum (bir kez): Repoyu git clone ile alın, .env, npm install.
	2-Günlük güncelleme: guncelle-ve-baslat.bat
		git pull → npm install → baslat.bat (sunucu + tarayıcı).
	3-Burada token.txt genelde gerekmez; GitHub’a push yapmıyorsanız Credential Manager veya HTTPS yeterli olabilir. (Pull 	için token.txt kullanımı sizin bat’larda tanımlıysa, sadece o PC’de gerekir.)
